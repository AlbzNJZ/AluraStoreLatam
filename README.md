# Análisis de rendimiento de tiendas

## Descripción del proyecto

Este proyecto tiene como objetivo analizar el desempeño de diferentes tiendas para determinar en cuál de ellas el Sr. Juan debería vender sus productos. Para ello se analizaron diferentes factores relacionados con las ventas, el comportamiento de los clientes y los costos asociados a las compras.

El análisis fue realizado utilizando Python en Google Colab, aplicando herramientas de análisis de datos y visualización para interpretar la información disponible.

---

## Objetivos

- Analizar los ingresos totales generados por cada tienda.
- Identificar las categorías de productos más y menos vendidas.
- Evaluar las calificaciones promedio de los clientes.
- Determinar cuáles productos tienen mayor demanda.
- Analizar la relación entre el precio de los productos y el costo de envío.

---

## Herramientas utilizadas

- Python
- Pandas
- Matplotlib
- Google Colab

Estas herramientas permitieron procesar los datos y generar diferentes gráficos para facilitar la interpretación de los resultados.

---

## Análisis realizado

Durante el proyecto se realizaron diferentes análisis y visualizaciones:

### Ingresos totales por tienda
Se compararon los ingresos generados por cada tienda para identificar cuál presenta mayor volumen de ventas.

### Ventas por categoría
Se analizaron las categorías de productos para identificar cuáles tienen mayor demanda entre los clientes.

### Calificación promedio de los clientes
Se evaluó el nivel de satisfacción de los clientes mediante las calificaciones otorgadas en cada tienda.

### Productos más vendidos
Se identificaron los productos que presentan mayor número de ventas.

### Relación entre precio y costo de envío
Se analizó cómo se relacionan los precios de los productos con los costos de envío para evaluar su impacto en la decisión de compra.

---

## Conclusión

A partir del análisis realizado se identificó la tienda con mejor desempeño general considerando los ingresos, la satisfacción de los clientes y los costos de envío. Con base en estos resultados se recomienda que el Sr. Juan venda sus productos en la tienda que presenta las mejores condiciones para generar mayores oportunidades de negocio.
